Как запустить программу на Lua

Способ первый. В терминале Linux последовательно введите команды:
curl -R -O http://www.lua.org/ftp/lua-5.4.4.tar.gz
sudo apt install -y nano
tar zxf lua-5.4.4.tar.gz
cd lua-5.4.4
make all test
cd src
nano privet.lua
Откроется редактор кода, в котором нужно ввести текст:
print("Привет, мир!")
Затем нажать Ctrl + S, Ctrl + X
Окно редактора закроется. Запустим программу, введя в терминал:
./lua privet.lua
На следующей строке появится сообщение Вашей первой программы на Lua:
Привет, мир!
Чтобы запускать программы командой lua, рекомендуется добавить путь к файлу lua из папки src в переменные окружения PATH и LUA_DIR в файле .bashrc, который находится в Вашей домашней папке Linux, для чего добавьте в конец текстового файла записи:
export PATH=$PATH:/путь/к/lua
export LUA_DIR=/путь/к/lua
Способ второй. Введите в Терминале Linux последовательно команды
sudo apt update
sudo apt upgrade
sudo apt install -y lua5.4
lua
print("Привет из интерпретатора Lua!")
После этого зажмите Ctrl + C, чтобы завершить работу с интерпретатором.
